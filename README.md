# 1 Swiggy Sales Dashboard

I’m excited to share my latest project: An interactive 'Swiggy Sales Dashboard' built entirely in Excel!

Using a dataset of over 197K orders, I designed this dashboard to track performance, identify trends, and understand customer preferences at a granular level.

Dashboard features:

✅ Interactive Filters: Easily slice data by Month, Category, and Restaurant.
✅ Multi-Dimensional Trends: In-depth analysis of daily, weekly, and monthly sales.
✅ Geographical Insights: State-level sales mapping for better market visibility.
✅ Product Mix: Clear breakdown of Veg vs. Non-Veg performance.

# Tools used: Excel (Power Pivot, Advanced Charting).

# 2 ☕ Coffee Shop Sales Dashboard

I’m excited to share my latest project: An interactive Coffee Shop Sales Dashboard built entirely in Excel!

Using a large-scale transactional dataset, I designed this dashboard to transform raw Point of Sale (POS) data into actionable business intelligence. This project focuses on optimizing operational efficiency and identifying key growth drivers.

Dashboard Features:
Dynamic Filtering: Interactive slicers allow users to drill down into data by Month and Day of the week for precise time-based analysis.

Peak Hour Optimization: Visualizes order patterns to identify critical rush hours, helping to align staffing levels with customer demand.

Product Performance: A granular breakdown of sales by category and top-performing products, providing clear insights into revenue drivers.

Geographical Analysis: Comparison of top store locations to evaluate performance across different branches.

# Tools Used:
Excel: Advanced Pivot Tables, Data Modeling, Slicers, and Dynamic Charting.
